Que es el desarrollo de aplicaciones Web?

El desarrollo de aplicaciones web es un proceso que implica la creación y mantenimiento de software destinado a ser ejecutado en un navegador web. Estas aplicaciones pueden variar en complejidad, desde simples páginas web interactivas hasta aplicaciones web empresariales complejas.
